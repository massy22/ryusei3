# Wi-Fi

Wi-Fiに関する情報を紹介していきます。

## ブラザーバンド
ワイヤレス通信で登録するか、WiFiでコードを互い登録するか、シナリオ中にオートブラザーと出会うと、ブラザーバンドを結ぶ事ができる。<br />
最大で六人まで登録可能で、結ぶ度にキズナリョクがＵＰする。一人当たりの最高値は150(オートブラザーは100)である。

データを更新することで、互いのレコードやベストコンボも更新され、相手のストーリーの進行具合によって＋されるキズナリョクも上昇していく。<br />
オートブラザーは再度話しかける事でデータが更新される。<br />
所持可能数を超えていなければレジェンドカードもその都度貰えるので、積極的に更新していこう。<br />

頻繁に更新していると、レジェンドカードを使わないとすぐに所持枠が一杯になってしまう。<br />
何度でももらえる物なのでケチケチしないで積極的に使ってＯＫ。<br />

なお、間違って(故意に)ブラザーを消してしまった場合相手のブラザーコードがわかっていたら再度入力する人とブラザーをくめる
