# キズナリョク
- キズナリョクが上がる条件は以下の通り

- ゲーム内でブラザーを組む(最大15人)

<br />ミソラ、ルナ、キザマロ、ゴン太の依頼をクリアする
<br />※最大900

- リアルブラザーを組む<br />※最大130×6=780(クリアスター番号1〜5を入手すれば130になる)
- リアルブラザーと同じチーム名にする<br />※最大40+10×6=100
- リアルブラザーと対戦をする(勝敗は関係ない)<br />※最大120(ブラザー1人でも可)

- 以上全てをこなすと1900になる（ブライノイズは1020まで）
