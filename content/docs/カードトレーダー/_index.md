# カードトレーダー
余ったカードの使い道はコレ<br />
<br />
コダマタウンのBIGWAVEと、ナンスカの入り口付近と、ヤエバのリゾートホテルの入り口と、ＴＫタワー（エンディング後）に設置されている。<br />
交換に必要なカードは、コダマタウンの方は三枚で、ナンスカの方は五枚で、ヤエバとＴＫタワーは十枚。<br />
それぞれ出るカードが違う。ＴＫタワーではメガカード（EX,SPを含む）も出る。<br />

また、今作ではトレーダー使用後にオートセーブされないので、内容を確認することも可能。ただし出るカードは決定しているのでリセットしても変わらない。<br />

トレーダーの上に電脳体の顔が見えている状態（ビジライザー使用か電波変換状態）で話しかけると、トレーダーチケットを使ったトレードが可能。<br />
トレーダーチケットは必要な枚数のカードの代わりに使えるだけの物で、バグピーストレーダーのような特殊な物ではない。<br />
また、電脳体が見えている状態でも、チケット使用選択時に「いいえ」を選択すると、トレーダーチケットを使わずにカードを使うことが可能。どちらでも出るカードは変わらない。<br />
