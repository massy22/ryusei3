# サブイベント
スターキャリアーを覗いて話しかけることで頼まれるサブイベントの一覧。<br />
全て解決するとレジェンドマスター・シンから、じげんのはざま２にプレゼントを置いたとメールが届く。<br />
全依頼解決で開く扉の奥には[ウォーロックの武器・マジックブレス](docs/ウォーロックの武器)がある。
