# 第2話

第2話の攻略です。

## 攻略チャート
10日後の夏休み…デンパクンのメールを見た後、HP+50/50入手

ロッポンドーヒルズのショッピングプラザに居る天地さんに話しかける

委員長の家へ

家へ帰って母さんに話しかけた後、パソコンを調べてパソコンの電波へ

ウイルスとの戦闘…VSメットリオG、バルカナGx2

戦闘後、ウェーブアウトし、寝る

バス停へ…

ヤエバリゾート付いたら、ホテルのカウンターにいる女の人に話しかける

グルメタウンの人やゴヨウダさんなどに話しかける

メールが来たら、カウンターの人に話しかける

スウィートルーム前へ行き、その後、電波変換してスウィートルームの中へ…

部屋の中のウェーブホールで、ウェーブアウト

その後、扉を調べて鍵をあける

グルメタウンのロープウェイ前まで行き、イベント後、リゾートエリアの入口へ

アイとの会話後、合計3つの雪だるまを調べて、スノーゴロンを倒していく（電波変換していると急な坂も登れるようになります）

リゾートホテルのロビーへ…

ルナの部屋に近づき、スウィートルームから出る

ゴンタとの会話後、もう一度部屋に入りヒゲンダンディーに話しかける

電波変換して、従業員専用フロアにいるお姉さんに話しかけマテリアルウェーブ おおぐいマシーン入手

ゴンタの前でおおぐいマシーンを使用する

VSオックスファイア

委員長の部屋に部屋に近づいた後、ソファーを調べて寝る

ゴンタと会話後、グルメタウンのロープウェイに乗り一番上へ

下へと滑って行き、グルメタウンへ

電波変化して雪山を登っていく

途中のミニゲームは十字キー左右を使ってクリアしていきましょう

VSイエティブリザード
