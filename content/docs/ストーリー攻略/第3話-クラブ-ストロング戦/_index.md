# 第3話

第3話の攻略です

# 攻略チャート
#br
コダマタウンの公園からスタート

ブラックエースの場合 強化アビリティ[エースPGM] を入手

レッドジョーカーの場合 強化アビリティ[ジョーカーPGM] を入手

キーアイテム[SPコード1]を入手

コダマタウンのWRに入りそのまま上へ行くとセキュリティがあるので<br />
ポップアップをタッチしセキュリティを開ける

ニホンのコスモウェーブ2からシーサーアイランドへ

シーサーじょう 奥にいるウィザードと話す

ゴン太と話す

やたいの電脳へ

やたいの電脳内に居る電波君と話す

サイバーアウト、ウェーブアウトをしてゴン太と話す

ビーチへ

パンツやぶりをする。内容はここでは言えない

ミソラと話す

いしがきの電脳へ

ウィルスと3連戦<br />
1戦目、アカシーサー、アカシーサーG<br />
2戦目、プルミンG、キルシャークG<br />
3戦目、アカシーサーG、キルシャークG、バルガンナー<br />

皆と話す

シーサーじょう 奥に居るストロングと話す

シーサーアイランドのWRにヒールウィザードが居るので話す<br />
話すと戦闘へ<br />
プルミン、メットリオ2、ヒールウィザード<br />

ビーチのWRにヒールウィザードが居るので話す<br />
話すと戦闘へ<br />
メットリオ2、キルシャーク、ヒールウィザード

何を選んでも以降のシナリオが変わるとかそんなことはないので好きなものを選ぶ<br />
しかしゴン太を選ぶとゴン太の電波変換の1シーンが見られるのでお得かもしれない

シーサーじょう へ

シーサーアイランドからWRへ行き<br />
シーサーじょうへ行きかんきょうシステムの電脳1へ

※落ちる茶色のWRは一度でも走りきると二度と落ちる事はない<br />
　 なお落ちると 電脳のならく1<br />4 へ行く<br />
　 変わったウィルスやミステリーデータは無いので見たい人は落ちると良い<br />
　 ジャンプはAボタン、ならくから戻ると落ちたWRの前に戻る<br />

かんきょうシステムの電脳3にて<br />
<strong>VS クラブ・ストロング</strong>

ビーチへ

<strong>VS ジャック・コーヴァス</strong><br />
10秒ほどで自動的に戦闘が終了する
