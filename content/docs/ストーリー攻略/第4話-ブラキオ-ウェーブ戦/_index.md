# 第4話

第4話の攻略です。

## 攻略チャート

委員長の家へ

外に出て、メールが来たらリゾートホテルのロビーにいる人（設定したオートブラザー）に話しかける

コダマタウンに帰って寝る

電波変換してアンテナを調べ、コダマタウンのスカイウェーブへ

特にトラップなどはないのでそのまま、ドンブラー村のスカイウェーブへ

ドンブラー村についたら、変な人に話しかけ、自動翻訳機入手

ゴヨウダさんや村の人に話しかける

ルナに話しかけた後、電波変換してピンクのデンパクンに話しかける

ウェーブロードをつかって離れ小島のデンパクンからフロートシューズをもらい、さっきのピンクのデンパクンにわたす

てんぼうだいで全ての人に話し掛ける

キザマロに話しかける

ドッシーの入り江でキザマロに話しかけた後、ルナに話しかける

事件発生、展望台へ

上の方へ行き、マテリアルウェーブ 潜水艦を入手

電波変換して、潜水艦を使用し、湖の底へ

ドンブラー湖1の電波では、イカダの前、ヒトデの前、カンムリの前を掘ればOK

ドンブラー湖2の電波では、倒れたイス、タコのいる坂を降りた洞窟、潜水艦の前、デンパクンの上の段、5個の岩の真ん中、を掘ればOK

VSブラキオウェーブ
