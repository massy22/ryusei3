# 第7話

第7話の攻略です

# 攻略チャート
#br
コダマタウン の おくじょう からスタート

コダマ小学校から出る

ゴン太、キザマロ、ミソラ、ルナと話す

WAXAニホンしぶ へ

左から1～3の ジャミングないぶ の入口が開く<br />
ヨイリーは ジャミングないぶ1 にいるので左の入口へ

ヨイリーの近くへ行くと<br />
VS コテツードG、リリッシュG

ヨイリーと話す

シーサーアイランド にある シーサーじょう へ

コダマタウン にある スバルのリビング へ

母親と話す

BAの場合ギガクラスバトルカード「ウィングブレード」を入手<br />
RJの場合ギガクラスバトルカード「Gメテオレーザー」を入手

スバルのへや へ行き、ベッドで寝る

スバルのリビング へ

オクダマスタジオ へ

左から4～6の ジャミングないぶ の入口が開く<br />
マモロウは4の左下のほうに、スズカは5の右上の方に居る

二人とも救出すると<br />
VS ウォーロックコピーW、ウォーロックコピーW

キーアイテム「ディーラーコード」を入手

コダマタウン のWR、または シーサーアイランド のWRから ニホンのコスモウェーブ1 へ

ニホンのコスモウェーブ2 へ

ムーン・ディザスターを倒していない場合は倒す

ニホンのコスモウェーブ2 の奥へ進むとセキュリティがあるのでディーラーコードを使い開く

奥へ進む

オービタルベース へ

流星コントロールそうちの電脳 へ

奥へ行くと<br />
<strong>VS スペード・マグネッツV2</strong><br />
※このスペード・マグネッツV2を倒すとコダマタウンのWRに居るスペード･マグネッツV2とは戦えなくなります

サイバーアウト、ウェーブアウトしハートレスのもとへ

ダイゴフォルダ を入手

WAXAニホンしぶ の しれい室 に行き、天地と話す

コダマタウン へ

キーアイテム「ダイゴコード」を入手

WAXAニホンしぶ へ

ビッグウェーブステーションを使い宇宙へ<br />
VS フレイマーG、スピラッシュG

ノイズウェーブ3 へ着くので ノイズウェーブ3 を抜け メテオサーバー1 へ

一定時間ごとにクリムゾンが襲い掛かってくるのでビーストスイングを使いながら避けて進む<br />
※クリムゾンと接触すると戦闘になるが、リザルト画面がないのでカードやゼニーは手に入らない

ダイゴコードを使いながら進もう<br />
※ダイゴコード使用時に自分が映るので自分が見えるように探してみよう

きおくのはへん を3つ集めたらノイズの壁があったところへ戻る

奥へ進むと<br />
<strong>VS ダイヤ・アイスバーンV2</strong><br />
※このダイヤ・アイスバーンV2を倒すとオクダマスタジオのWRに居るダイヤ・アイスバーンV2とは戦えなくなります

メテオサーバー2 へ

きおくのはへん を4つ集めたらノイズの壁があったところへ戻る

奥へ進むと<br />
<strong>VS クラブ・ストロングV2</strong><br />
※このクラブ・ストロングV2を倒すとシーサーじょうのWRに居るクラブ・ストロングV2とは戦えなくなります

メテオサーバー3 へ

きおくのはへん を5つ集めたらノイズの壁があったところへ戻る

奥へ進むと<br />
<strong>VS ジャック・コーヴァスV2</strong><br />
※このジャック・コーヴァスV2を倒すとおくじょうのWRに居るジャック・コーヴァスV2とは戦えなくなります

さらに奥へ進むと<br />
<strong>VS クイーン・ヴァルゴV2</strong><br />
※このクイーン・ヴァルゴV2を倒すとしれい室のWRに居るクイーン・ヴァルゴV2とは戦えなくなります

奥へ進もうとするとセーブを要求される<br />
セーブをし次のエリアへ進むともう戻ることはできない

メテオサーバー4最深部へ行くと<br />
<strong>VS クリムゾン・ドラゴン</strong>

エンディングへ･･･
