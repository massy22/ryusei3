# 基本要項
4話のノイズチェンジイベント以降、前回のノイズチェンジ発生後（キャンセル含む）Gウィルス戦に勝利した回数に比例して上昇していく。<br />発生率は以下の通り
|Gウィルス戦|ノイズチェンジの発生確率|
|---|---|
|1回目|33％|
|2回目|50％|
|3回目|70％|
|4回目|80％|
|5回目|100％|

チェンジできる種類はソフトのバージョンごとに異なる。<br />別バージョンのノイズにチェンジする確率は1/8で、さらに5種類の中からランダムで選択される。

|ブラックエース|レッドジョーカー|
|---|---|
|リブラ<br />コーヴァス<br />キャンサー<br />ジェミニ<br />オヒュカス|キグナス<br />オックス<br />ヴァルゴ<br />クラウン<br />ウルフ|

ブライノイズは、3回目以降のノイズチェンジでリアルブラザーとライバルノイズを登録していない場合、低確率でなることができる。<br />
（※初回のイベント変身時にはならない）

    電波障壁はバスター1発で解除されてしまうため、あくまで気休め程度であるが目の前にダブルストーンを設置することにより、防御面を強化することも可能。<br />
    １ターンに一度電波障壁は復活する。攻撃を当てるまで電波障壁は見えないので忘れないように。<br />
    チャージショットはそのままなのでアビリティや武器によって様々なカスタマイズができる。<br />
    極めて高い火力を持つ反面、リアルブラザー登録ができないため絶対的なキズナ力が低く、装備できるアビリティが限られる。<br />
    また、ブラザーとのレゾン共有によるファイナライズの強化も出来ない点にも注意。<br />
    シークレットサテライトサーバーを有効活用しよう。<br />
    ノイズ解除されないため、ノイズ率が上昇してもファイナライズせず、そのまま戦うほうが得策である。
