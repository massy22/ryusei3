# Ｗスロット連動イベント
エグゼシリーズをＷスロットした常態でゲームを開始するとサテラポリスから「謎のテキストデータ」に関するメールが届き、<br />ゲーム内でキーアイテムが見つかるようになります。<br />
5つの日記を全て入手し、全て読むと熱斗からメールが届き、ウォーロック装備「[エグゼブラスター](docs/ウォーロックの武器)」が手に入ります。
### 対応ソフト
ロックマンエグゼ（アドバンス版）<br />
ロックマンエグゼバトルチップGP
